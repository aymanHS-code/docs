## Communication Flows

<Steps>
  <Step title="User Authentication">
    User → Auth → Handler → Database → Logger
  </Step>
  <Step title="Device Control">
    User → AI → Handler → Devices → Provider → Device → Logger
  </Step>
  <Step title="Payment Processing">
    User → AI → Handler → Services (Payments) → Gateway → Email → Logger
  </Step>
</Steps>
