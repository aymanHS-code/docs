---
title: "Send OTP"
api: "POST /api/send-otp"
authMethod: none
openapi: "/openapi-specs/Roomi Auth.json POST /api/send-otp"
---

Send OTP to user phone number or unit admin phone.

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |


## Body

```json
{
  "phone": "+971501234567",
  "unit_id": "unit-123"
}
```
