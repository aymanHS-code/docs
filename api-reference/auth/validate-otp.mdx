---
title: "Auth - Validate OTP"
api: "POST /v1/auth/validate-otp"
authMethod: "key"
---

Validate a one-time password (OTP) to authenticate a user.

<Note>
  Sign your request with AWS SigV4 and include these headers in your request

  - Authorization
  - X-Amz-Content-Sha256
  - X-Amz-Date
  - X-Unit-ID
  - X-User-ID
</Note>

```json
// Example body
{
  "phone": "+971501234567",
  "otp": "123456",
  "pin": "pin-id-from-send-otp"
}
```
