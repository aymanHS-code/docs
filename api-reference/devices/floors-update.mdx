---
title: "Update Floor Name"
api: "PUT /api/floors/{floorName}"
authMethod: none
openapi: "/openapi-specs/Roomi Devices.json PUT /api/floors/{floorName}"
---

Update the floor name across all rooms that currently have the specified floor name.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| floorName | string | Yes | Current floor name to update |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |
| X-Unit-ID | Yes | Unit identifier |
| X-User-ID | Yes | User identifier |


## Body

```json
{
  "newFloorName": "Main Floor"
}
```
