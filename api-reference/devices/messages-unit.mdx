---
title: "Messages for Unit"
api: "GET /api/devices/messages/unit"
authMethod: none
openapi: "/openapi-specs/Roomi Devices.json GET /api/devices/messages/unit"
---

Retrieve all device-related messages and notifications for a specific unit.

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |
| X-Unit-ID | Yes | Unit identifier |
| X-User-ID | Yes | User identifier |
