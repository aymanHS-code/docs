---
title: "Create Room"
api: "POST /api/rooms"
authMethod: none
openapi: "/openapi-specs/Roomi Devices.json POST /api/rooms"
---

Create a new room in the unit with specified name, floor, and optional device assignments.

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |
| X-Unit-ID | Yes | Unit identifier |
| X-User-ID | Yes | User identifier |


## Body

```json
{
  "roomName": "Home Office",
  "roomFloor": "First Floor",
  "deviceMes": ["device-789","device-101"]
}
```
