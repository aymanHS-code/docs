---
title: "List Devices"
api: "GET /v1/devices"
authMethod: none
openapi: "/openapi-specs/Roomi Devices.json GET /api/devices"
---

Retrieve all devices for a specific unit with their status and capabilities.

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |
| X-Unit-ID | Yes | Unit identifier |
| X-User-ID | Yes | User identifier |


<ResponseExample>

```json Response
[
  {
    "name": "Living Room Lights",
    "unitId": "unit-12345",
    "room": "Living Room",
    "brand": "lifesmart",
    "status": "online",
    "type": "light"
  }
]
```

</ResponseExample>
