---
title: "Devices - Control"
api: "POST /v1/devices/control"
authMethod: "none"
---

Control multiple devices with a single request.

<Note>
  Sign your request with AWS SigV4 and include these headers in your request

  - Authorization
  - X-Amz-Content-Sha256
  - X-Amz-Date
  - X-Unit-ID
  - X-User-ID
</Note>

```json
// Example body
{
  "user_id": "user-123",
  "unit_id": "unit-456",
  "commands": [
    { "device_id": "device-123", "action": "turn_on" }
  ]
}
```
