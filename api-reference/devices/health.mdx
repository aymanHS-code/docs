Check the health status of the Devices service.

<Note>
  Sign your request with AWS SigV4 and include these headers in your request

  - Authorization
  - X-Amz-Content-Sha256
  - X-Amz-Date
  - X-Unit-ID
  - X-User-ID
</Note>
