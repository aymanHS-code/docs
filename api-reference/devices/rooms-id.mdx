---
title: "Room by ID"
api: "GET /api/rooms/{roomId}"
authMethod: none
openapi: "/openapi-specs/Roomi Devices.json GET /api/rooms/{roomId}"
---

Retrieve a specific room by its ID with detailed device information.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| roomId | string | Yes | Room identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |
| X-Unit-ID | Yes | Unit identifier |
| X-User-ID | Yes | User identifier |



<ResponseExample>

```json Response
{
  "roomId": "LIVINGROOM",
  "roomName": "Living Room",
  "roomFloor": "Ground Floor",
  "deviceMes": [
    "device-123",
    "device-456"
  ],
  "devices": [
    {
      "me": "device-123",
      "name": "Living Room Light"
    },
    {
      "me": "device-456",
      "name": "Living Room Fan"
    }
  ]
}
```

</ResponseExample>
