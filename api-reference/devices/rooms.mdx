title: Rooms
api: "GET /api/rooms"

openapi: "/openapi-specs/Roomi Devices.json GET /api/rooms"

Retrieve all rooms for a unit with their associated device information.

<Note>
  Requires `X-Unit-ID` and `X-User-ID` plus SigV4 headers.
</Note>

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |
| X-Unit-ID | Yes | Unit identifier |
| X-User-ID | Yes | User identifier |
