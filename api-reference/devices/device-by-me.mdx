Retrieve a specific device using its ME identifier.

<Note>
  Sign your request with AWS SigV4 and include these headers in your request

  - Authorization
  - X-Amz-Content-Sha256
  - X-Amz-Date
  - X-Unit-ID
  - X-User-ID
</Note>
