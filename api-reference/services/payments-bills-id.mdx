---
title: "Bill by ID"
api: "GET /api/payments/bills/{billId}"
authMethod: none
openapi: "/openapi-specs/Roomi Services.json GET /api/payments/bills/{billId}"
---

Retrieve a specific bill by its ID.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| billId | string | Yes | Bill identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |



<ResponseExample>

```json Response
{
  "billId": "bill-123",
  "amount": 150.5,
  "description": "Monthly maintenance fee",
  "dueDate": "2024-02-15",
  "status": "pending",
  "createdDate": "2024-01-15T10:00:00Z"
}
```

</ResponseExample>
