api: "PUT /api/routines/{routineId}"
authMethod: none

Update an existing routine.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| routineId | string | Yes | Routine identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |


## Body

```json
{
  "name": "Updated Morning Workout",
  "description": "Updated daily morning exercise routine",
  "schedule": "Daily at 6:30 AM",
  "activities": ["Push-ups","Squats","Running","Yoga"],
  "duration": 75,
  "isActive": true
}
```
