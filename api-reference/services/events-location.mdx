---
title: "Events by Location"
api: "GET /api/events/location/{locationId}"
authMethod: none
openapi: "/openapi-specs/Roomi Services.json GET /api/events/location/{locationId}"
---

Retrieve events for a specific location.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| locationId | string | Yes | Location identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |



<ResponseExample>

```json Response
[
  {
    "eventId": "event-123",
    "locationId": "location-456",
    "eventName": "Community Meeting",
    "eventDate": "2024-01-15T10:00:00Z",
    "description": "Monthly community gathering"
  }
]
```

</ResponseExample>
