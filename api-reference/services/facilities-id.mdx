---
title: "Facility by ID"
api: "GET /api/facilities/{facilityId}"
authMethod: none
openapi: "/openapi-specs/Roomi Services.json GET /api/facilities/{facilityId}"
---

Retrieve a specific facility with its details and available slots.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| facilityId | string | Yes | Facility identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |

  "facilityId": "facility-123",
  "facilityName": "Swimming Pool",
  "rating": 4.5,

  "availableSlots": [
