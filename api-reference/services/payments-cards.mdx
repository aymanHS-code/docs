---
title: "Payment Cards"
api: "POST /api/payments/cards"
authMethod: "none"
---

Add a new payment card to the user account. Use GET /api/payments/cards to list cards and DELETE /api/payments/cards/{cardId} to delete.

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |


## Body

```json
{
  "cardNumber": "4111111111111111",
  "expiryDate": "12/25",
  "cvv": "123",
  "cardHolderName": "John Doe"
}
```
