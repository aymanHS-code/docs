api: "GET /api/payments/bills/{billId}/status"

openapi: "/openapi-specs/Roomi Services.json GET /api/payments/bills/{billId}/status"

Retrieve the payment status of a specific bill.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| billId | string | Yes | Bill identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |



<ResponseExample>

```json Response
{
  "billId": "bill-123",
  "status": "pending",
  "lastPaymentDate": "2024-01-10T14:30:00Z",
  "nextDueDate": "2024-02-15"
}
```

</ResponseExample>
