---
title: "Booking by Slot"
api: "GET /api/facilities/bookings/{slotId}"
authMethod: none
openapi: "/openapi-specs/Roomi Services.json GET /api/facilities/bookings/{slotId}"
---

Retrieve detailed booking information for a specific slot.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| slotId | string | Yes | Slot identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |



<ResponseExample>

```json Response
{
  "bookingId": "booking-123",
  "slotId": "slot-456",
  "facilityId": "facility-123",
  "facilityName": "Swimming Pool",
  "bookedBy": "unit-789",
  "bookingDate": "2024-01-15T10:00:00Z"
}
```

</ResponseExample>
