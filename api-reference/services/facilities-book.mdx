---
title: "Book Facility Slot"
api: "POST /api/facilities/book"
authMethod: none
openapi: "/openapi-specs/Roomi Services.json POST /api/facilities/book"
---

Book a specific time slot for a facility.

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |


## Body

```json
{
  "facilityId": "facility-123",
  "slotId": "slot-456",
  "unitId": "unit-789"
}
```
