# Roomi AI API

Roomi AI Assistant API for processing messages, checking task statuses, and platform health.\
All requests require **AWS SigV4 Authentication**.

> **AWS SigV4 Authentication**
>
> - `Authorization` — AWS4-HMAC-SHA256 signature
> - `X-Amz-Content-Sha256` — SHA256 hash of the request body
> - `X-Amz-Date` — ISO8601 timestamp (e.g., `20230101T120000Z`)
>
>   \
>   **Region:** `me-central-1`\
>   **Service:** `execute-api`

## Get Task Status

<ApiPlayground
  method="GET"
  path="/task/{task_id}/status"
  baseUrl="https://ai-{env}-ae.roomi-services.com"
  headers={{
  "x-unit-id": "unit-12345",
  "x-user-id": "user-67890"
}}
>
  <Param name="task_id" type="string" required>
    The ID of the task to check.
  </Param>

  <Response status={200}>
    ```json
    {
      "status": "completed",
      "message": "Task finished successfully",
      "error": null
    }
    ```
  </Response>

  <Response status={422}>
    ```json
    {
      "detail": [
        {
          "loc": ["path", "task_id"],
          "msg": "Invalid UUID format",
          "type": "value_error"
        }
      ]
    }
    ```
  </Response>
</ApiPlayground>

## Process Message

<ApiPlayground
  method="POST"
  path="/roomi"
  baseUrl="https://ai-{env}-ae.roomi-services.com"
  headers={{
  "Content-Type": "application/json",
  "x-unit-id": "unit-12345",
  "x-user-id": "user-67890"
}}
>
  <Request>
    ```json
    {
      "user_id": "user-67890",
      "unit_id": "unit-12345",
      "unit_name": "Living Room Speaker",
      "transcript": "Turn on the lights"
    }
    ```
  </Request>

  <Response status={200}>
    ```json
    {
      "response": "Turning on the living room lights",
      "background_tasks": [],
      "query_type": "DEVICE_CONTROL",
      "query_types": ["DEVICE_CONTROL"],
      "transcript_parts": ["lights", "on"]
    }
    ```
  </Response>

  <Response status={422}>
    ```json
    {
      "detail": [
        {
          "loc": ["body", "unit_id"],
          "msg": "field required",
          "type": "value_error.missing"
        }
      ]
    }
    ```
  </Response>
</ApiPlayground>
