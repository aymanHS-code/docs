---
title: "AI - Health"
api: "GET /v1/health"
authMethod: "none"
---

Check the health status of the AI service.

<Note>
  Sign your request with AWS SigV4 and include these headers in your request

  - Authorization
  - X-Amz-Content-Sha256
  - X-Amz-Date
</Note>

```json
// Example response
{
  "status": "healthy",
  "message": "AI service is operational",
  "timestamp": "2024-01-15T10:00:00Z"
}
```
