---
title: "AI - Task Status"
api: "GET /v1/task/{task_id}/status"
authMethod: "none"
---

Check the status of a background task.

<Note>
  Sign your request with AWS SigV4 and include these headers in your request

  - Authorization
  - X-Amz-Content-Sha256
  - X-Amz-Date
</Note>

```json
// Example response
{
  "task_id": "task-789",
  "status": "completed",
  "message": "All requested actions completed successfully",
  "progress": 100
}
```
