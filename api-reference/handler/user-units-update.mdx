---
title: "Update User Unit"
api: "PATCH /user-units/{user_id}/{unit_id}"
authMethod: "none"
---

Update a user-unit relationship.

## Path Parameters


| Name | Type | Required | Description |
| --- | --- | --- | --- |
| user_id | string | Yes | User identifier |
| unit_id | string | Yes | Unit identifier |


## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |


## Body

```json
{
  "access_type": "member",
  "user_type": "tenant"
}
```
