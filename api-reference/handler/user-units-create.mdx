---
title: "Create User Unit"
api: "POST /user-units"
authMethod: "none"
---

Create a new user-unit relationship.

## Headers


| Name | Required | Description |
| --- | --- | --- |
| Authorization | Yes | AWS4-HMAC-SHA256 signature |
| X-Amz-Content-Sha256 | Yes | SHA256 hash of the request body |
| X-Amz-Date | Yes | ISO8601 timestamp |


## Body

```json
{
  "user_id": "user-123",
  "unit_id": "unit-456",
  "access_type": "admin",
  "user_type": "owner"
}
```
