## Overview

This quick start guide will help you integrate the Roomi SDK into your application in just a few steps. You'll learn how to install the SDK, configure authentication, and make your first API calls.

## Prerequisites

Before you begin, ensure you have:

- **Roomi Account**: Access to Roomi services with API credentials
- **AWS Credentials**: Access key ID and secret access key for authentication
- **Development Environment**: Node.js (v16+) or Python (v3.8+)

## Step 1: Installation

### JavaScript/TypeScript

```bash
npm install @roomi/sdk
```

### Python

```bash
pip install roomi-sdk
```

## Step 2: Configuration

Set up your environment variables:

```bash
# .env file
AWS_ACCESS_KEY_ID=your_access_key_here
AWS_SECRET_ACCESS_KEY=your_secret_key_here
ROOMI_UNIT_ID=unit-12345
ROOMI_USER_ID=user-67890
ROOMI_REGION=me-central-1
```

## Step 3: Initialize the SDK

### JavaScript/TypeScript

```javascript
import { RoomiSDK } from '@roomi/sdk';

const roomi = new RoomiSDK({
  region: 'me-central-1',
  accessKeyId: process.env.AWS_ACCESS_KEY_ID,
  secretAccessKey: process.env.AWS_SECRET_ACCESS_KEY,
  unitId: process.env.ROOMI_UNIT_ID,
  userId: process.env.ROOMI_USER_ID
});
```

### Python

```python
from roomi_sdk import RoomiSDK
import os

roomi = RoomiSDK(
    region='me-central-1',
    access_key_id=os.getenv('AWS_ACCESS_KEY_ID'),
    secret_access_key=os.getenv('AWS_SECRET_ACCESS_KEY'),
    unit_id=os.getenv('ROOMI_UNIT_ID'),
    user_id=os.getenv('ROOMI_USER_ID')
)
```

## Step 5: Your First Integration

### Example 1: User Authentication

```javascript
// Send OTP to user's phone
const otpResponse = await roomi.auth.sendOTP({
  phone: '+971501234567',
  unit_id: 'unit-123'
});

console.log('OTP sent:', otpResponse.pinId);

// Validate OTP and get user token
const authResponse = await roomi.auth.validateOTP({
  phone: '+971501234567',
  otp: '123456',
  pin: otpResponse.pinId
});

console.log('User authenticated:', authResponse.user);
```

### Example 2: Device Control

```javascript
// Get all devices in the unit
const devices = await roomi.devices.getAllDevices();
console.log('Available devices:', devices);

// Control a specific device
await roomi.devices.controlDevices({
  user_id: 'user-123',
  unit_id: 'unit-456',
  data: {
    devices: [
      {
        deviceId: 'device-456',
        targetValue: '100',
        actionType: 'set_brightness',
        actionId: 1
      }
    ]
  }
});

console.log('Device controlled successfully');
```

### Example 3: AI Assistant

```javascript
// Process a natural language command
const aiResponse = await roomi.ai.processMessage({
  user_id: 'user-123',
  unit_id: 'unit-456',
  unit_name: 'Apartment 101',
  transcript: 'Turn on the living room lights'
});

console.log('AI Response:', aiResponse.response);
console.log('Query Type:', aiResponse.query_type);
```

### Example 4: Facility Booking

```javascript
// Get available facilities
const facilities = await roomi.services.facilities.getAllFacilities();
console.log('Available facilities:', facilities);

const booking = await roomi.services.facilities.book({

console.log('Facility booked:', booking);
```

## Step 6: Error Handling

Implement proper error handling for production use:

```javascript
async function safeApiCall(apiFunction, ...args) {
  try {
    const result = await apiFunction(...args);
    return { success: true, data: result };
  } catch (error) {
    console.error('API Error:', error.message);
    
    // Handle specific error types
    if (error.status === 401) {
      console.error('Authentication failed - check credentials');
    } else if (error.status === 429) {
      console.error('Rate limit exceeded - implement backoff');
    } else if (error.status >= 500) {
      console.error('Server error - retry later');
    }
    
    return { success: false, error: error.message };
  }
}

// Usage
const result = await safeApiCall(roomi.devices.getAllDevices);
if (result.success) {
  console.log('Devices:', result.data);
} else {
  console.log('Error:', result.error);
}
```

## Step 7: Complete Example

Here's a complete example that demonstrates multiple SDK features:

```javascript
import { RoomiSDK } from '@roomi/sdk';

class RoomiSmartHome {
  constructor() {
    this.roomi = new RoomiSDK({
      region: 'me-central-1',
      accessKeyId: process.env.AWS_ACCESS_KEY_ID,
      secretAccessKey: process.env.AWS_SECRET_ACCESS_KEY,
      unitId: process.env.ROOMI_UNIT_ID,
      userId: process.env.ROOMI_USER_ID
    });
  }

  async initialize() {
    try {
      // Test all services
      await this.roomi.auth.health();
      await this.roomi.devices.getAllDevices();
      await this.roomi.ai.health();
      await this.roomi.services.health();
      
      console.log('‚úÖ All services connected successfully');
      return true;
    } catch (error) {
      console.error('‚ùå Initialization failed:', error.message);
      return false;
    }
  }

  async authenticateUser(phone) {
    try {
      // Send OTP
      const otpResponse = await this.roomi.auth.sendOTP({
        phone,
        unit_id: process.env.ROOMI_UNIT_ID
      });

      // In a real app, you'd prompt the user for OTP
      const userOTP = '123456'; // This would come from user input

      // Validate OTP
      const authResponse = await this.roomi.auth.validateOTP({
        phone,
        otp: userOTP,
        pin: otpResponse.pinId
      });

      console.log('User authenticated:', authResponse.user);
      return authResponse;
    } catch (error) {
      console.error('Authentication failed:', error.message);
      throw error;
    }
  }

  async controlDevices(deviceCommands) {
    try {
      await this.roomi.devices.controlDevices({
        user_id: process.env.ROOMI_USER_ID,
        unit_id: process.env.ROOMI_UNIT_ID,
        data: { devices: deviceCommands }
      });

      console.log('Devices controlled successfully');
    } catch (error) {
      console.error('Device control failed:', error.message);
      throw error;
    }
  }

  async processVoiceCommand(transcript) {
    try {
      const response = await this.roomi.ai.processMessage({
        user_id: process.env.ROOMI_USER_ID,
        unit_id: process.env.ROOMI_UNIT_ID,
        unit_name: 'Smart Home',
        transcript
      });

      console.log('AI Response:', response.response);
      return response;
    } catch (error) {
      console.error('AI processing failed:', error.message);
      throw error;
    }
  }

  async getFacilities() {
    try {
      const facilities = await this.roomi.services.facilities.getAllFacilities();
      console.log('Available facilities:', facilities);
      return facilities;
    } catch (error) {
      console.error('Failed to get facilities:', error.message);
      throw error;
    }
  }
}

// Usage
async function main() {
  const smartHome = new RoomiSmartHome();
  
  // Initialize
  const initialized = await smartHome.initialize();
  if (!initialized) {
    console.error('Failed to initialize Roomi SDK');
    return;
  }

  // Authenticate user
  await smartHome.authenticateUser('+971501234567');

  // Control devices
  await smartHome.controlDevices([
    {
      deviceId: 'device-456',
      targetValue: '100',
      actionType: 'set_brightness',
      actionId: 1
    }
  ]);

  // Process voice command
  await smartHome.processVoiceCommand('Turn on the living room lights');

  // Get facilities
  await smartHome.getFacilities();
}

main().catch(console.error);
```
