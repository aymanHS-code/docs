## Core Services

### AI Service - The Brain of the System

The AI service is the central intelligence that orchestrates all other services. It processes natural language commands, makes intelligent decisions, and coordinates actions across the entire platform.

**Key Capabilities:**

- Natural language processing and understanding
- Context-aware conversations and decision making
- Orchestration of all other services
- Background task management
- Voice synthesis and processing

**Base URL:** `https://ai-{environment}-{region}.roomi-services.com/v1/`

### Devices Service - Smart Device Abstraction

The Devices service provides a unified interface for controlling any smart device, regardless of the underlying provider. It abstracts away the complexity of different smart home protocols and manufacturers.

**Key Capabilities:**

- Universal device control interface
- Multi-provider support (Philips Hue, SmartThings, etc.)
- Device discovery and management
- Room and floor organization
- Real-time device status monitoring

**Base URL:** `https://devices-{environment}-{region}.roomi-services.com/v1/`

### Handler Service - System Communication Hub

The Handler service manages all database operations and inter-service communication. It acts as the central nervous system, handling data persistence and coordinating between different services.

**Key Capabilities:**

- Database operations and data management
- Inter-service communication routing
- Event handling and message queuing
- System state management
- Data synchronization

**Base URL:** `https://handler-{environment}-{region}.roomi-services.com/v1/`

## Support Services

### Auth Service - Private Authentication System

Our invite-only authentication system provides secure user management and access control for the Roomi platform.

**Key Capabilities:**

- Invite-only user registration
- OTP-based phone verification
- UAE Pass integration
- JWT token management
- User session handling

**Base URL:** `https://auth-{environment}-{region}.roomi-services.com/v1/`

### Logger Service - CloudWatch Integration

The Logger service provides centralized logging and monitoring capabilities, integrating with AWS CloudWatch for comprehensive system observability.

**Key Capabilities:**

- Structured logging to CloudWatch
- Performance monitoring
- Error tracking and alerting
- Audit trail management
- System health monitoring

**Base URL:** `https://logger-{environment}-{region}.roomi-services.com/v1/`

### Services Platform - Third-Party Integration

The Services platform handles all third-party integrations and external API communications, providing a unified interface for property developer and user-specific services.

**Key Capabilities:**

- Third-party service integration
- Property developer API management
- User-specific service handling
- Payment processing
- Event and routine management

**Base URL:** `https://services-{environment}-{region}.roomi-services.com/v1/`
