## Core Services

### AI Service - The Brain of the System

The AI service is the central intelligence that orchestrates all other services. It processes natural language commands, makes intelligent decisions, and coordinates actions across the entire platform.

<CardGroup cols={2}>
  <Card title="Natural Language Processing" icon="message-captions">
    Process and understand user commands in multiple languages
  </Card>
  <Card title="Service Orchestration" icon="chart-network">
    Coordinate actions across all platform services
  </Card>
  <Card title="Context Management" icon="brain">
    Maintain conversation context and user preferences
  </Card>
  <Card title="Background Tasks" icon="clock">
    Handle long-running operations asynchronously
  </Card>
</CardGroup>

### Devices Service - Smart Device Abstraction

The Devices service provides a unified interface for controlling any smart device, regardless of the underlying provider. It abstracts away the complexity of different smart home protocols and manufacturers.

<CardGroup cols={2}>
  <Card title="Universal Control" icon="bolt-lightning">
    Standardized commands for any smart home device
  </Card>
  <Card title="Multi-Provider Support" icon="layer-group">
    Support for Lifesmart, Philips Hue, SmartThings, Tuya, and more
  </Card>
  <Card title="Device Discovery" icon="magnifying-glass">
    Automatic detection and registration of new devices
  </Card>
  <Card title="Real-time Status" icon="bell-on">
    Live device status monitoring and updates for all device types on the platform
  </Card>
</CardGroup>

### Handler Service - System Communication Hub

The Handler service manages all database operations and inter-service communication. It acts as the central nervous system, handling data persistence and coordinating between different services.

  </Card>
</CardGroup>
